Basic Structure

  • A competition titled e.g., "Kode gubbz gaming night" comprises one or several games (e.g., Bowling, StarCraft).
  • Each game contains several matches in order for all teams to face each other.
  • One match always consists of two teams competing against each other.
  • In these matches, every team competes against each other exactly once.

Score Calculation for Each Match

Time-based Games:

  • If the game's scoring is based on time (e.g., fastest lap in a car game), the team with the least cumulative time wins the match. This is calculated from the scores of all players in the team.
  • The team with the least total time receives 100 points, followed by 90 points for the next least time, and so on.

Score-based Games:

  • If the game's scoring is determined by a score (e.g., total pins in bowling), the team with the highest cumulative score wins the match. This is calculated from the scores of all players in the team.
  • The team with the highest total score gets 100 points, followed by 90 points for the next highest score, and so on.

Total Team Score Calculation (Leaderboard)

  • The overall score is derived from the cumulative total score of each team across all games.
  • In instances where teams have uneven member counts, the score for each match will be determined by average for all teams.
  • For example, if one team has 3 members and another has 2, scores will be averaged based on 3 players for the first team and 2 players for the second.

Time-based Games:

  • The team with the least combined cumulative time across a game earns 100 points. The next least time gets 90 points, and so forth.

Score-based Games:

  • The team with the highest combined cumulative score across a game receives 100 points, while the next highest score is awarded 90 points, and so on.

Total Player Score Calculation (Leaderboard)

  • The overall score is derived from the cumulative total score of each player across all games.
  • In instances where teams have uneven member counts, the score for each match will be determined by average for all players.
  • For example, if one team has 3 members and another has 2, scores will be averaged based on 3 players for the first team and 2 players for the second.

    • Time-based Games:

    • The player with the combined least total time across a game is awarded 100 points, with the next least time getting 90 points, and so forth.

    Score-based Games:

    • The player with the combined highest total score in a game secures 100 points. The subsequent highest scorer gets 90 points, continuing in this pattern.
