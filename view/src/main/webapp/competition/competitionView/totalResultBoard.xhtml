Basic structure


One competition (Kode gubbz gaming night) consist of one or several games (Bowling, StarCraft etc....)
Each game has several matches where all teams face each other exactly one time.

Score calculation for each match


If a games score is calculated based on time (for example fastest lap in a car game), the team with the total least time wins the match (Calculated on all players in the team)
If a games score is calculated based on score (for example total score in bownling), the team with the total most score wins the match (Calculated on all players in the team)

The team with the least total time in a match gets 100 points
The team with the next least total time in a match gets 90 points and so on...

The team with the most total score in a match gets 100 points
The team with the next most total score in a match gets 90 points and so on...

Total Team score calculation (This leaderboard)


The total score is calculated on the total score for each team in every game

If a games score is calculated based on time, the team with the least total time in a game gets 100 points
The team with the next least total time in a game gets 90 points and so on...

If a games score is calculated based on score, the team with the most total score in a game gets 100 points
the team with the next most total score in a game gets 90 points and so on...

Total Player score calculation (This leaderboard)


The total score is calculated on the total score for each player in every game

If a player score is calculated based on time, the player with the least total time in a game gets 100 points
The player with the next least total time in a game gets 90 points and so on...

If a games score is calculated based on score, the player with the most total score in a game gets 100 points
the player with the next most total score in a game gets 90 points and so on...
